Bienvenido al código fuente de mi portfolio personal. Una aplicación Single Page Application (SPA) ultra-rápida y bilingüe construida para exhibir mis proyectos, experiencia profesional y habilidades como Licenciado en Sistemas y Desarrollador Full Stack.

## ✨ Características Principales (Wow Factors)

Este proyecto fue desarrollado poniendo un fuerte énfasis en el rendimiento, la accesibilidad y un diseño moderno (UI/UX):

- **Arquitectura Moderna**: Desarrollado con Angular 19 utilizando Standalone Components y Signals para el manejo reactivo del estado.
- **Sistema Bilingüe Inmediato**: Internacionalización (i18n) ligera creada desde cero con Angular Signals (ES/EN) sin dependencias externas, logrando cambios de idioma instantáneos sin recargar la página.
- **Diseño Glassmorphism**: Estética moderna con tarjetas translúcidas, desenfoque de fondo y paleta de colores oscuros profundos.
- **Efecto 3D Parallax (Tilt)**: Directiva custom que calcula la posición del mouse para aplicar inclinación 3D en tiempo real a las tarjetas, otorgando una inmersiva sensación de profundidad.
- **Cursor Interactivo Customizado**: Reemplazo del cursor nativo por un sistema dual (punto y anillo expansivo) animado a 60fps usando trigonometría y `requestAnimationFrame`.
- **Preloader Animado**: Pantalla de carga (Splash screen) inicial que oculta la carga de recursos y revela el contenido de manera elegante.

## 🛠️ Stack Tecnológico

- **Framework:** Angular 19 (Standalone)
- **Lenguaje:** TypeScript
- **Estilos:** Vanilla CSS (CSS Variables, Flexbox, CSS Grid)
- **Animaciones:** CSS Keyframes, Transform 3D, Intersection Observer API
- **Deployment:** Vercel (CI/CD Automático)

## 🚀 Despliegue Local

Para correr este proyecto en tu máquina local:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/FelipeSchenberger/portfolio.git
   ```
2. Instalar las dependencias:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   ng serve
   ```
4. Abrir en el navegador: `http://localhost:4200`
